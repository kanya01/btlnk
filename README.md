# Bottleneck: Working Memory Demonstrator

Bottleneck is a single-screen, local-first browser game designed to demonstrate the limitations of human working memory and the profound effect of segmented delivery (pacing) on recall performance. By interacting with a delivery slider, the player directly experiences the cognitive difference between "all at once" and "bit by bit" presentation.
